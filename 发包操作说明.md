
首次发包前的操作
1.  在HyFrame根目录下执行命令：npm i;
2.  在HyFrame\projects\frame目录下执行命令：npm i;
3.  点开HyFrame目录下的theme.js文件（这个文件傻傻的，不点开好像代码执行的时候，不会引用到npm i安装上的东西）;
4.  修改HyFrame\projects\frame/package.json的版本号;
5.  打包操作：在HyFrame根目录下执行命令 npm run build，成功之后会在HyFrame根目录下生成一个dist文件夹;
6.  修改npm默认地址：npm set registry=https://reg.haiyisec.com:8443/repository/npm-host/;
7.  登录npm操作：执行命令 npm login ,按照提示输入账号、密码、邮箱（邮箱随意填写）;
    账号：test 密码：Haiyisec@123
8.  发包操作：进入HyFrame/dist/frame的目录下执行命令：npm publish;

正常发包操作
1.  修改HyFrame\projects\frame/package.json的版本号;
2.  在HyFrame根目录下执行命令：npm run build;
3.  进入HyFrame/dist/frame的目录下执行命令：npm publish;
